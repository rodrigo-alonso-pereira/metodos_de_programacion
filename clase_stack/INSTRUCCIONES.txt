El programa funciona con un archivo de lectura en texto plano de nombre "stack.in"

El formato de este archivo debe ser el siguiente:

La primera línea debe indicar el número de instrucciones a ejecutar
Las siguientes líneas deben indicar la instrucción en mayúscula
Las instrucciones soportadas son:

Suma: SUM
Resta: RES
Multiplicación: MUL
División:DIV
Módulo: MOD
Raíz cuadrada: RAI
E^x: EXP
Potencia: POT
Extraer elemento: POP
Insertar elemento: PUSH, esta instrucción debe acompañarse por un espacio y el número a insertar

Este programa trabaja con números enteros, en caso de algún resultado con parte decimal, será truncado.

FINALMENTE, LAS INSTRUCCIONES SE LEERÁN DE ABAJO HACIA ARRIBA

****************

Ejemplo de entrada para la operación 1 + 3 * 4 / 2

7
SUM
PUSH 1
DIV
MUL
PUSH 3
PUSH 4
PUSH 2
